### 1.3 启动 API 服务

LLaMA-Factory 提供 OpenAI 兼容的 API 接口，可以直接用 `openai` SDK 调用。

**坑 1：vllm 装不上**
之前装 vllm 时依赖冲突，跳过。改用 huggingface backend（推理慢一点但稳定）。

**坑 2：`--port` 和 `--api_port` 都不被识别**

```bash
# ❌ 试过这些都报错或被忽略
llamafactory-cli api --port 6006 ...
llamafactory-cli api --api_port 6006 ...
```

查源码发现端口是通过**环境变量** `API_PORT` 设置的。

**坑 3：为什么用 6006 端口？**
AutoDL 默认只把 6006 端口（TensorBoard 端口）映射到公网，其他端口要走 SSH 隧道。所以 API 服务直接占用 6006。

**最终启动命令**：

```bash
cd /root/autodl-tmp/ft_cs

nohup bash -c 'source .venv/bin/activate && \
    API_PORT=6006 CUDA_VISIBLE_DEVICES=0 llamafactory-cli api \
    --model_name_or_path ./models/Qwen2.5-7B-Instruct \
    --adapter_name_or_path ./checkpoints/ft_cs_v1 \
    --template qwen \
    --finetuning_type lora \
    --infer_backend huggingface' > api.log 2>&1 &

# 查看启动日志
tail -f api.log

# 看到 "Uvicorn running on http://0.0.0.0:6006" 就是成功了
# 记下 PID 方便后面 kill
ps aux | grep llamafactory-cli
```

**验证 API 服务（在服务器本地）**：

```bash
curl http://localhost:6006/v1/chat/completions \
    -H "Content-Type: application/json" \
    -d '{
        "model": "ft_cs_v1",
        "messages": [
            {"role": "system", "content": "你是一个电商客服助手..."},
            {"role": "user", "content": "我想查我的订单"}
        ],
        "temperature": 0
    }'
```

返回合法 JSON 即 API 服务正常。

### 1.4 AutoDL 公网地址 ≠ 可直接调 API

AutoDL 给 6006 端口的公网地址：`https://u925462-8415-dd7c76ee.bjb1.seetacloud.com:8443`

⚠️ **这个地址只能用浏览器打开（TensorBoard 模式），不能用 httpx/openai SDK 直接 POST**。原因是 AutoDL 反代对长连接和 POST 请求有限制，会报 `httpx.RemoteProtocolError`。

**解决方案：SSH 隧道**

把服务器的 6006 端口映射到本机 6006 端口，本机直接调 `http://localhost:6006`，相当于服务跑在本机一样。这是最稳的方式，**也是明天要做的关键操作**。

### 1.5 服务器端评估（直接本地评估）

为了不依赖 SSH 隧道，先在服务器上跑一遍评估，验证模型效果。

评估脚本逻辑：
1. 读取 `ft_eval.json` 545 条数据
2. 每条数据取 `conversations[1]['value']` 作为用户输入，`conversations[2]['value']` 作为期望输出
3. 调用 `http://localhost:6006/v1/chat/completions`
4. 统计**合法 JSON 率**（输出能 `json.loads` 解析）和**轨道准确率**（task/knowledge/chitchat 三选一是否预测正确）

**关键代码片段**（已保存为 `scripts/deploy/ft_evaluate_local.py`）：

```python
import httpx, json

system = '你是一个电商客服助手，负责分析用户消息并输出 TurnPlan JSON。...'

with open('llamafactory/ft_eval.json') as f:
    data = json.load(f)

total = len(data)
valid_json = 0
track_correct = 0

for i, sample in enumerate(data):
    msgs = sample['conversations']
    user_msg = msgs[1]['value']
    expected = json.loads(msgs[2]['value'])
    
    try:
        resp = httpx.post('http://localhost:6006/v1/chat/completions', json={
            'model': 'ft_cs_v1',
            'messages': [
                {'role': 'system', 'content': system},
                {'role': 'user', 'content': user_msg}
            ],
            'temperature': 0
        }, timeout=30)
        output = resp.json()['choices'][0]['message']['content']
        parsed = json.loads(output)
        valid_json += 1
        
        # 轨道准确率
        exp_track = 'task' if expected.get('task') else ('knowledge' if expected.get('knowledge') else 'chitchat')
        out_track = 'task' if parsed.get('task') else ('knowledge' if parsed.get('knowledge') else 'chitchat')
        if exp_track == out_track:
            track_correct += 1
    except:
        pass
    
    if (i+1) % 50 == 0:
        print(f'进度: {i+1}/{total}, 合法JSON率: {valid_json/(i+1)*100:.1f}%, 轨道准确率: {track_correct/(i+1)*100:.1f}%')

print(f'最终：合法JSON率 {valid_json/total*100:.2f}%, 轨道准确率 {track_correct/total*100:.2f}%')
```

**当前进度**：50/545，合法 JSON 率 100%，轨道准确率 100%。

⏳ 等全量 545 条跑完，截图保存评估结果。
